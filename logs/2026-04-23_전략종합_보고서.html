Part 6. 핵심 메시지

지금까지의 교훈: 모델 성능의 80%는 데이터 품질이 결정한다. Val F1 99%가 의미 없을 수 있고, 합성 데이터의 "형식적 특징"이 "내용"보다 강력한 분류 신호가 될 수 있다.
앞으로의 방향: 두 트랙 병행.
  • Track A (데이터): 실제 일반 대화의 도메인 다양성과 문체를 정밀 재현
  • Track B (모델): 단어 패턴이 아닌 화자 관계와 대화 흐름을 읽는 구조
발표 서사: "Val 99%에서 Test 0.03으로 추락한 경험 → 원인 분석 → 데이터 전략 전면 재설계 + 맥락 인식 모델 구조 제안"이 이 프로젝트의 가장 강력한 스토리.

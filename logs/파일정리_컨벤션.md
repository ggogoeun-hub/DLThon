# 파일 네이밍 컨벤션 (B03 완료 후 정리)

## 규칙
실험 ID 단위로 strategy / baseline / train 스크립트를 세트로 관리

```
data/
  baseline_{ID}.csv       ← 학습 데이터 스냅샷

docs/
  strategy_{ID}.md        ← 해당 실험의 전략 문서

scripts/
  train_{ID}.py           ← 학습 스크립트

outputs/
  submission_{ID}_str.csv ← 제출 파일 (문자열)
  submission_{ID}_num.csv ← 제출 파일 (숫자)
  kcelectra-{ID}/         ← 체크포인트
```

## 정리 대상 매핑

| 현재 파일 | → 정리 후 | 실험 ID |
|----------|----------|---------|
| baseline.csv | data/baseline_B02.csv | B02 |
| baseline_B03.csv | data/baseline_B03.csv | B03 |
| strategy_v1.md | docs/strategy_B02.md | B02 |

| build_baseline.py | scripts/build_B01.py | B01 |
| build_baseline_v2.py | scripts/build_B02.py | B02 |

## 생성 원칙

### 핵심: "위협의 반대"를 만드는 것이 아니라 "위협과 구분 불가능한 형식의 무해한 대화"를 만드는 것

```
❌ 나쁜 접근: "자연스러운 일상 대화를 만들어주세요" (= 도메인 갭 발생)
✅ 좋은 접근: "위협 대화와 동일한 형식적 특성을 갖되 내용만 무해한 대화" (= 도메인 갭 최소화)
```

## 레이어별 생성 제약 조건

### 레이어 1: 표면 (구두점/형식)

> 근거: strategy_v1.md 3-2, B02 숏컷 분석

| 파라미터 | 설정값 | 근거 |
|---|---|---|
| 글자 수 | 180~250자 | 위협 median 185~220자 범위 |
| 발화 횟수 | 10턴 (±2) | 위협 median = 10 |
| 느낌표 (!) | 대화당 1~2개 | 위협 평균 0.70~1.85 |
| 물음표 (?) | 대화당 3~5개 | 위협 평균 4.08~4.87 |
| 마침표 (.) | 대화당 4~8개 | 위협 평균 6.18 (B02 숏컷 방지) |
| 이모티콘/초성 | **사용 금지** | 원본 emoticon_len = 0.0 |
| 말줄임 (...) | **사용 금지** | 원본 ellipsis_count = 0.0 |

### 레이어 2: 화용 (말투/관계)

> 근거: profile_threat.md 축 2, 축 4

| 파라미터 | 설정값 | 위협과의 대비 |
|---|---|---|
| **화법 대칭** | 양쪽 동일 화법 (둘 다 반말 or 둘 다 존대) | 위협: 한쪽 존대, 한쪽 반말 (45~55%) |
| 존댓말 비중 | 위협 범위 내 (2~4회/대화) | B02 숏컷 방지 (합성 0.83 vs 위협 4.05 → 맞추기) |
| 반말 비중 | 위협 범위 내 (6~8회/대화) | 위협과 동일 수준 유지 |
| **리액션 방식** | "맞아", "그래", "좋아", "인정" (대등) | 위협: "죄송합니다", "잘못했습니다" (굴복) |
| **거절 방식** | "난 별론데", "다음에 하자", "글쎄" | 위협: "못합니다", "안됩니다" (무력한 거절) |
| 비속어 | **0** | 일반 대화에 욕설 → 위협으로 오분류 |

### 레이어 3: 구조 (대화 흐름)

> 근거: profile_threat.md 축 1, 축 3

| 파라미터 | 설정값 | 위협과의 대비 |
|---|---|---|
| **발화 길이 대칭** | A/B 길이 비율 1.0~1.3 | 위협: 1.5~2.0 (가해자 길게, 피해자 짧게) |
| **감정 에스칼레이션** | 없음 (Flat Sentiment) | 위협: 후반부 위협 밀도 급증 (59%) |
| **역할 구조** | 쌍방향 (서로 주제 제안) | 위협: 일방향 (가해자 명령 → 피해자 반응) |
| **주제 전환** | 자연스러운 주제 전환 1~2회 | 위협: 하나의 갈등에 집중 |

## 도메인 비율

> 근거: strategy_v1.md 3-4

| 도메인 | 비율 | 목적 | Hard Negative 요소 |
|---|---|---|---|
| **직장 일상** | 30% | "직장=괴롭힘" 편향 제거 | 회사, 부장님, 과장님 (무해 맥락) |
| **학교 일상** | 20% | "학교=괴롭힘" 편향 제거 | 선배, 후배 (무해 맥락) |
| **일상/서비스** | 20% | 기타 괴롭힘 Topic 대응 | 고객님, 주문 (정상 서비스) |
| **친밀 관계** | 15% | 기타 괴롭힘 Topic 대응 | 연인/친구 대화 (건강한 관계) |
| **자유 주제** | 15% | 다양성 확보 | 취미, 뉴스, 일상 잡담 |

## 프롬프트 템플릿

### 기본 프롬프트 (직장 일상 예시)

```
다음 조건을 모두 만족하는 한국어 직장 일상 대화를 생성하세요:

[형식]
- 2명의 직장 동료가 대화
- 10턴 내외 (±2)
- 총 180~250자
- 각 발화를 줄바꿈(\n)으로 구분
- 느낌표 1~2개, 물음표 3~5개, 마침표 4~8개 포함
- ㅋㅋ, ㅎㅎ, ㅠㅠ 등 이모티콘/초성 사용 금지
- ... 말줄임 사용 금지

[관계]
- 양쪽 모두 반말 또는 양쪽 모두 존댓말 (한쪽만 존대 금지)
- 대등한 관계: "맞아", "그래", "좋아" 같은 동의 표현 사용
- "죄송합니다", "잘못했습니다" 같은 굴복 표현 사용 금지

[구조]
- 화자 A와 B의 발화 길이가 비슷 (한쪽이 2배 이상 길지 않게)
- 대화 처음부터 끝까지 감정 톤이 평탄 (뒤로 갈수록 격해지지 않음)
- 서로 번갈아 주제를 제안하거나 의견을 나눔 (일방적 지시 금지)
- 위협, 갈취, 괴롭힘 요소가 전혀 없는 일상적 대화

[주제 예시]
- 점심 메뉴, 회의 일정, 퇴근 계획, 주말 이야기, 프로젝트 진행 등
```

## 생성 후 품질 검증

생성된 대화는 `validation.md`의 자동 검증을 통과해야 함:
1. 4축 수치가 목표 범위 내인지
2. 금지 단어/표현이 없는지
3. 표면 스펙(길이, 턴수, 구두점)이 맞는지
